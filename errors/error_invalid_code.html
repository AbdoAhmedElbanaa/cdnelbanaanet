خطأ بكود التفعيل

  1. تم التلاعب بكود التفعيل
  2. تم استخدام النظام بإحدى الطرق غير الرسميه
  3. كود التفعيل غير مطابق للموجود فى قاعدة التفعيل
  4. عدم وجود كود تفعيل
